# Uzay Simülasyonu Projesi (Space Simulation Project)

#Dönem Projesi

## Proje Hakkında
C programlama dili ile geliştirilen bu konsol uygulaması, kullanıcıdan (bilim insanından) alınan verilerle 9 farklı fizik deneyini Güneş Sistemi'ndeki gezegenler üzerinde simüle eder.

## Özellikler
* Pointer aritmetiği ile dizi erişimi.
* Modüler fonksiyon yapısı.
* Ternary operatörü ile hata yönetimi ve mutlak değer dönüşümü.
* Girdi doğrulama (Input Validation).
